# Getting Started

To create a project use the Solid cli:

```bash
pnpm create solid
```

:::warning[_]
We recommend using the `bare` template with `typescript`. Others have not been tested, but should work with some modifications.
:::

```
│
◇  Project Name
│  solidbase-project
│
◇  Is this a SolidStart project?
│  Yes
│
◇  Which template would you like to use?
│  bare
│
◇  Use Typescript?
│  Yes
│
```
